## 채택 체크리스트

- [ ] 식별자 + 정체성 + 말투 5샘플 채워짐
- [ ] 시스템 프롬프트 3~5줄 작성됨
- [ ] `dialogue_pairs.jsonl` 에 60+ 페어 작성됨 (이 디렉터리)
- [ ] 메타 단어 검사 통과 (`pytest tests/test_training.py`)
- [ ] CJK 비율 ≥ 0.5
- [ ] LoRA v1 학습 + 비교 harness 통과 → `models/ours/qwen-{persona_id}-v1/`
